# Full Stack AI Habit Tracker App
 AI Habit Tracker | MERN, React 19, Node.js, Express, MongoDB, JWT, Gemini AI  Built a full-stack habit tracking platform with secure JWT authentication, protected routes, and role-safe API access. Designed REST APIs for habits, logs, stats, and AI insights; implemented streaks, heatmaps, weekly summaries, and analytics. Integrated Google Gemini AI to generate weekly reports, habit suggestions, streak recovery plans, and Q&A insights. Developed a responsive React UI with Vite, Tailwind CSS, Axios, and Recharts for interactive data visualization. Implemented environment-based configuration, CORS handling, and robust error responses for production readiness.

## Run Locally

Backend (API):

```bash
cd backed
npm install
npm run dev
```

Frontend (UI):

```bash
cd frontend/ai-habit-tracker-ui-boilerplate-code
npm install
npm run dev
```
